# 에이전트 규칙

- **언어:** 사용자와의 대화는 **한국어**(답변·질문·설명). 코드·식별자는 관례에 따라 영문 허용.
- 이 프로젝트는 **Vite + React**(SPA), 라우팅은 **React Router**. Next.js API(`next/*`, App Router, Server Components)는 사용하지 않는다.
- 디렉터리 구조는 **Feature-Sliced Design**을 따른다. `CLAUDE.md`의 레이어 규칙과 import 방향을 지킨다.
- 스타일은 **Tailwind CSS v4**. 정적 자산은 `public/`, 경로는 `/파일명`으로 참조한다.
