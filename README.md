# 당비비 (dangbibi)

소방·경찰 공무원 근무 패턴 확인 웹 서비스. **Vite + React**, 구조는 **Feature-Sliced Design**을 따릅니다.

## 시작하기

```bash
pnpm install
pnpm dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 을 엽니다.

## 스크립트

| 명령 | 설명 |
|------|------|
| `pnpm dev` | 개발 서버 |
| `pnpm build` | 타입 검사 + 프로덕션 빌드 |
| `pnpm preview` | 빌드 결과 미리보기 |
| `pnpm lint` | ESLint |

에이전트·컨벤션은 루트 `CLAUDE.md`, `AGENTS.md`를 참고하세요.
